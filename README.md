# FDU中文文本分类语料库预处理工具

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个简单易用的工具，专门用于处理 [复旦大学中文文本分类语料库](https://www.keenkit.com/2020/08/fudan-chinese-text-classification-corpus/](https://gitcode.com/open-source-toolkit/283c1))（通常称为 FDU 语料库）。  
它可以帮助你完成两个最繁琐的预处理步骤：

- **转码**：将原始 ANSI（GBK/GB2312）编码的文本文件转换为 UTF‑8 编码。
- **合并**：将所有文件合并成一个单一文件，每篇文档占一行，文档之间用空行分隔，方便后续机器学习/自然语言处理任务直接使用。

> 🔗 关于 FDU 语料库的详细说明，请参考原作者链接。本工具**不包含原始语料库**，你需要自行获取并遵守其使用许可。

---

## ✨ 功能特点

- **一键转码**：自动识别并转换 ANSI 编码（GBK/GB2312）到 UTF‑8。
- **智能合并**：将所有转码后的文件合并为一个文件，每篇文档一行，文档间以空行分隔。
- **纯前端实现**（HTML 版本）：无需安装任何软件，直接在浏览器中运行，保护你的数据隐私。
- **命令行版本**（Python 脚本）：适合批量处理和自动化集成。

---

## 🚀 使用方法

### 选项：HTML 网页工具（推荐快速试用）

1. 下载本仓库中的 `预处理工具包.html` 文件（或直接在线打开）。
2. 在浏览器中打开该 HTML 文件，你将看到一个简洁的界面：
   - **步骤 1**：点击“选择文件”，**一次性选择所有需要处理的 ANSI 编码文本文件**（可以多选）。
   - **步骤 2**：点击“转码并合并”，工具会自动将每个文件转换为 UTF‑8，并按文件名排序后合并。
   - **步骤 3**：处理完成后，点击“下载结果”保存生成的 `.txt` 文件。

> ⚠️ 注意：由于浏览器安全限制，请确保所选文件均为 ANSI 编码（如 GBK/GB2312）。如果文件较大，建议分批处理或使用命令行版本。

#### 使用说明

1. 本应用输入为若干ANSI编码的.txt文件
2. 本应用输出为
   - **选项一**：若干UTF-8编码的.txt文件
   - **选项二**：由若干UTF-8编码的.txt文件组成的.zip压缩包
   - **选项三**：一个合成之后，每个.txt文件占一行，之间空一行的，UTF-8编码的.txt文件
